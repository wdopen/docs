# 标题
在wikidot中，要创建一个标题，请使用以下语法:
<table>
<tr>
<th>wd语法</th>
<th>效果</th>
</tr>
  <tr>
    <td><code>+ 一级标题</code></td>
    <td><h1>一级标题</h1></td>
  </tr>
  <tr>
    <td><code>++ 二级标题</code></td>
    <td><h2>二级标题</h2></td>
  </tr>
  <tr>
    <td><code>+++ 三级标题</code></td>
    <td><h3>三级标题</h3></td>
  </tr>
  <tr>
    <td><code>++++ 四级标题</code></td>
    <td><h4>四级标题</h4></td>
  </tr>
  <tr>
    <td><code>+++++ 五级标题</code></td>
    <td><h5>五级标题</h5></td>
  </tr>
  <tr>
    <td><code>++++++ 六级标题</code></td>
    <td><h6>六级标题</h6></td>
  </tr>
</table>
